# 🚀 LangChain AI Agent Suite

+ A personal collection of LangChain-based AI agents built for self-learning and hands-on exploration of LLMs, agents, and real-world AI automation.

## 🧠 Overview

+ This repository contains multiple AI agents built independently as part of my self-study journey in LangChain and LLM application development.

+ Each project explores different real-world use cases like research, planning, learning, and automation using LLM-powered agents.

+ 🔬 AI Research Agent
  + Web + document-based intelligent research assistant

+ 📚 Learning Roadmap Agent
  + Generates structured learning paths for any skill

  + Builds optimized study schedules based on goals and time

  + AI-powered itinerary and travel planner

+ 🧠 Researcher Agent
  + Multi-source research and summarization agent
